## 🗒️ Project
청년층에서 유행하고 있는 유형 테스트에서 아이디어를 얻어, 각자의 여행 취향에 맞는 부산 여행지 추천 및 여행지 인근 식당/숙소/관광지 추천 서비스

## 🖥️ Development Environment
- node.js
- javascript
- ejs
- html
- css
## 🛠️ Tool
- VS Code
## 🔎 API
- Chat GPT
    - Text
    - Image
- Naver Service
  - Search (blog/local)
- Naver Cloud
  - Maps (Static Map)
  - Places (Geocoding/Reverse Geocoding)
## 🔗 Run
```
node server.js
```
